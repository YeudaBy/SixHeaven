# [SixHeaven](https://sixheaven.yeudaby.com/) - Kashrut Wait Timer 🥩 ✨ 🥛

אפליקציית **SixHeaven** היא אפליקציית אנדרואיד מודרנית, נקייה וקלה לשימוש למעקב אחר שעות המתנה בין בשר לחלב (ולהיפך), המעוצבת בסטנדרטים הגבוהים ביותר של UX/UI. האפליקציה נבנתה כדי להתאים לכל מנהג (Minhag) בצורה גמישה ואלגנטית.

> ## דיסקליימר: אפליקציה זו, כולל דף הנחיתה, הלוגו׳אים ואפילו דף README זה, נבנתה ב99% על ידי כלי Ai שונים. אני לא לוקח שום אחריות על מה שקרה/קורה/יקרה בריפוזיטורי הזה
-----

## ✨ תכונות עיקריות (Features)

  * **ניהול זמני המתנה מותאמים:**
      * **בשרי לחלבי:** תמיכה ב-6 שעות, 5 שעות, ו-"רוב שש" (**5:31**).
      * **חלבי לבשרי:** תמיכה בחצי שעה, שעה או 6 שעות.
      * **זמן מותאם אישית:** אפשרות להגדרה ידנית של כל זמן המתנה.
  * **אינטגרציה מלאה למערכת:**
      * **Persistent Notifications:** טיימר רץ בשורת ההתראות תמיד.
      * **Exact Alarms:** התראות מדויקות ברגע שהזמן מסתיים.
      * **Widgets:** ווידג'טים מעוצבים למסך הבית ולמסך הנעילה.
      * **Quick Settings Tile:** הפעלת הטיימר ישירות מהגדרות המהירות של אנדרואיד.
  * **פרטיות וקוד פתוח:**
      * ללא פרסומות, ללא מעקב מידע (No Tracking), וקוד פתוח לחלוטין.

## 🎨 קו עיצובי (Design Language)

הפרויקט משתמש בשפה עיצובית ייחודית המשלבת:

  * **צבעים:** בורדו עמוק (`#800A2C`), תכלת שמיים (`#38BDF8`) וזהב חם.
  * **סגנון:** Glassmorphism (זכוכית חלבית) ושימוש ב-Blur Blobs ליצירת עומק.
  * **טיפוגרפיה:** שימוש בגופנים נקיים ומודרניים לקריאות מקסימלית.

## 🛠 טכנולוגיות (Tech Stack)

  * **Language:** Kotlin / Java (או React Native/Flutter - בהתאם למה שבחרת עם קלוד).
  * **UI:** Material 3 / Tailwind CSS (אם זה ווב).
  * **Architecture:** Clean Architecture עם תמיכה ב-Background Services לניהול הטיימר.
